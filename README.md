# AKASA Portfolio Website

## Files you need to add
Put these image files in the same folder as the HTML files:
- `titleimage.jpg`
- `1proj.jpg`
- `2proj.jpg`
- `3proj.jpg`
- `4proj.jpg`

## How to run in VS Code
1. Open this folder in VS Code.
2. Make sure all your images are in the same folder.
3. Open `index.html` in your browser.
4. For easiest preview, use the VS Code **Live Server** extension.

## How to upload to GitHub
1. Create a new GitHub repository.
2. Upload all files in this folder.
3. Commit and push.
4. Enable GitHub Pages in the repo settings.
5. Set the source branch to `main` and root folder `/`.

Your homepage file is `index.html`.
